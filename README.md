# Leetcode-Sync
A tool to automate the uploads in my DSA_Journey repository.<br><br>

Features:<br>
  Fetches accepted LeetCode submissions<br>
  Organizes by difficulty (Easy, Medium, Hard)<br>
  Uses LeetCode question IDs in filenames<br>
  Includes topic tags in filenames<br>
  Stores multiple accepted solutions as Method 1, Method 2, ...<br>
  Supports multiple languages in one file<br>
  Automatically commits and pushes changes to GitHub<br>
  Avoids duplicate solutions<br>
  Retries failed network requests<br>
